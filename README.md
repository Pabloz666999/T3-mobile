# 📱 Tugas Praktikum Mobile: Form Identitas Mahasiswa

Aplikasi Android sederhana yang mengimplementasikan penggunaan XML Layout (termasuk bersarang/Nested Layout), ScrollView, dan View Validation menggunakan Kotlin.

## ✨ Fitur Aplikasi

1. **Input Data Dinamis:** Memiliki form untuk Nama (EditText), Jenis Kelamin (RadioGroup/RadioButton), dan Hobi (CheckBox).
2. **Validasi Form:**
   - Mencegah input Nama kosong dengan menampilkan pesan *error* langsung di dalam kolom `EditText`.
   - Menampilkan peringatan berupa `Toast` jika Jenis Kelamin belum dipilih.
3. **Pemrosesan Data:** Mengambil seluruh nilai dari komponen input dan menampilkannya kembali ke layar (pada `TextView` hasil) dalam format yang rapi setelah tombol ditekankan.
4. **Desain UI/UX:**
   - Menggunakan *custom drawable* untuk *background*, tombol, dan *container* agar antarmuka lebih menarik.
   - Menggunakan `ScrollView` agar aplikasi tetap responsif dan bisa di-scroll pada layar berukuran kecil.

## 🛠️ Teknologi yang Digunakan

* **Bahasa Pemrograman:** Kotlin
* **Minimum SDK:** API 24 (Nougat)
* **UI/Layouting:** XML (LinearLayout, ScrollView)
* **IDE:** Android Studio
